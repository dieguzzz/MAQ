# ⚡ Comandos Rápidos para Clonar el Proyecto

## 🚀 Clonar y Configurar (Una Línea)

```bash
# 1. Clonar repositorio
git clone https://github.com/dieguzzz/MAQ.git && cd MAQ

# 2. Instalar dependencias
flutter pub get

# 3. Verificar configuración
flutter doctor
```

## 🔄 Sincronizar Cambios

### Desde la Nueva Computadora
```bash
# Actualizar desde GitHub
git pull

# Ver cambios pendientes
git status
```

### Subir Cambios
```bash
# Agregar cambios
git add .

# Hacer commit
git commit -m "Descripción de cambios"

# Subir a GitHub
git push
```
