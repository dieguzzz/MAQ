Documentacion de Modulos

Guia completa de funcionalidades de MetroPTY

App Movil

Mapa y Estaciones

lib/screens/home/

Pantalla principal de la app. Muestra el mapa interactivo del metro con todas las estaciones, estado en tiempo real de cada estacion (aglomeracion, estado operacional), y la ubicacion del usuario. Permite seleccionar estaciones para ver detalles y hacer reportes rapidos.

Mapa interactivo Ubicacion GPS Estado en tiempo real Seleccion de estacion
App Movil

Sistema de Reportes

lib/screens/reports/

Permite a los usuarios reportar el estado de las estaciones y tiempos de espera de trenes. Incluye reportes simplificados (rapidos), reportes de tiempo de tren (ETA), confirmacion de llegadas, y un historial completo de reportes realizados.

Reporte rapido ETAs de trenes Confirmacion de llegada Historial
App Movil

Planificador de Rutas

lib/screens/routes/

Calcula la mejor ruta entre dos estaciones del metro. Muestra el tiempo estimado de viaje, las estaciones intermedias, y si se requiere transbordo entre lineas. Utiliza el servicio de calculo de rutas para encontrar caminos optimos.

Ruta optima Transbordos Tiempo estimado
App Movil

Gamificacion

lib/screens/gamification/ + lib/screens/leaderboards/

Sistema de puntos, niveles, badges y leaderboards que incentiva la participacion. Los usuarios ganan puntos al hacer reportes, suben de nivel, desbloquean insignias, y compiten en rankings semanales y globales.

Puntos por reportes Sistema de niveles Badges/Insignias Leaderboards
App Movil

Perfil de Usuario

lib/screens/profile/

Pantalla de perfil del usuario con estadisticas personales, badges obtenidos, historial de puntos, nivel actual y progreso. Incluye configuracion de preferencias y acceso a ajustes de la cuenta.

Estadisticas Badges Configuracion
App Movil

Autenticacion

lib/screens/auth/

Sistema de login con soporte para Google Sign-In y modo invitado. Maneja la sesion del usuario, permisos, y el flujo de onboarding para nuevos usuarios. Incluye upgrade de cuenta invitado a cuenta completa.

Google Sign-In Modo invitado Onboarding
App Movil

Premium y Monetizacion

lib/screens/premium/ + lib/services/ad_service.dart

Gestion de suscripciones premium y sistema de anuncios (AdMob). Los usuarios premium tienen experiencia sin anuncios y funcionalidades exclusivas. Incluye rewarded ads para ganar puntos extra.

AdMob integrado Plan premium Rewarded ads
App Movil

Aprendizaje

lib/screens/learning/

Modulo educativo que ayuda a los usuarios a conocer las estaciones del metro. Incluye datos curiosos, informacion de cada estacion, y un sistema de conocimiento que trackea lo que el usuario ha aprendido.

Info de estaciones Knowledge tracking Datos curiosos
Dashboard

Overview (KPIs)

dashboard/js/stats.js

Panel principal del dashboard con KPIs en tiempo real: total de usuarios, reportes del dia, estaciones activas, y trenes virtuales. Muestra graficas de tendencias y metricas clave para monitorear la salud del sistema.

KPIs en vivo Tendencias Tiempo real
Dashboard

Gestion de Estaciones

dashboard/js/stations.js

Monitoreo completo de todas las estaciones del metro. Visualiza estado operacional, nivel de aglomeracion, numero de reportes por estacion, y alertas activas. Permite filtrar y buscar estaciones especificas.

Monitoreo Alertas Filtros
Dashboard

Trenes Virtuales

dashboard/js/trains.js

Sistema de trenes virtuales que agrega reportes de usuarios para estimar la posicion y tiempo de llegada de trenes. Muestra el estado de cada tren virtual, su direccion, y el consenso de multiples reportes.

Trenes virtuales Consenso ETAs
Dashboard

Gestion de Reportes

dashboard/js/reports.js

Vista administrativa de todos los reportes generados por usuarios. Permite filtrar por tipo, estacion, fecha, y estado. Incluye detalles completos de cada reporte con metadata como nivel de confianza y verificacion.

Filtros avanzados Detalles Confianza
Dashboard

Gestion de Usuarios

dashboard/js/users.js

Administracion de usuarios registrados. Muestra rankings, estadisticas por usuario, nivel de actividad, puntos acumulados, y badges obtenidos. Permite ver el detalle de cada usuario y su historial de participacion.

Rankings Administracion Actividad
Dashboard

Analitica y Graficas

dashboard/js/analytics.js + time_analysis.js

Modulo de analitica avanzada con graficas interactivas (Chart.js). Analisis de tendencias de uso, patrones por hora y dia, distribucion de reportes por estacion, y metricas de precision de los ETAs.

Graficas interactivas Analisis temporal Precision ETAs
Servicio

Firebase / Firestore

lib/services/firebase_service.dart

Servicio central de conexion con Firebase. Gestiona la comunicacion con Firestore (base de datos en tiempo real), autenticacion de usuarios, y Cloud Messaging para notificaciones push.

Firestore Auth Push notifications
Servicio

Servicio de Ubicacion

lib/services/location_service.dart + background_location_service.dart

Gestiona la ubicacion GPS del usuario en primer y segundo plano. Determina la estacion mas cercana, calcula distancias, y permite el tracking de ubicacion en background para deteccion automatica de estaciones.

GPS Background Estacion cercana
Servicio

Motor de Simulacion

lib/services/train_simulation_service.dart + metro_simulator_service.dart

Motor que simula el movimiento de trenes basado en reportes de usuarios. Agrega ETAs, calcula posiciones virtuales, y mantiene el estado de los trenes. Incluye un simulador para desarrollo y testing.

Simulacion Agregacion ETAs Modo dev
Servicio

Validacion y Confianza

lib/services/report_validation_service.dart + confidence_service.dart + accuracy_service.dart

Sistema multi-capa para validar reportes de usuarios. Calcula el nivel de confianza basado en la reputacion del usuario, consistencia con otros reportes, proximidad GPS, y frecuencia de envio.

Validacion Score confianza Precision GPS
Servicio

Notificaciones

lib/services/notification_service.dart + alert_service.dart

Sistema de notificaciones push via Firebase Cloud Messaging. Envia alertas sobre estado de estaciones, llegadas de trenes, logros desbloqueados, y recordatorios de reporte. Soporta navegacion directa desde la notificacion.

Push FCM Alertas Deep linking
Servicio

Sistema de Puntos y Niveles

lib/services/gamification_service.dart + level_service.dart + points_reward_service.dart

Motor de gamificacion que gestiona puntos, niveles, y recompensas. Calcula puntos por tipo de reporte, aplica bonificaciones, maneja el progreso de nivel, y otorga badges por logros especificos.

Puntos Niveles Recompensas
